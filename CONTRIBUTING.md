# Contributing to APIX Gateway

Welcome! Thanks for your interest in contributing.

## Development Setup

```bash
# Clone the repo
git clone https://github.com/sagar0163/apix-gateway.git
cd apix-gateway

# Install dependencies
npm install

# Run in development
npm run dev

# Run tests
npm test
```

## Making Changes

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Make changes
4. Commit: `git commit -m "feat: add new feature"`
5. Push to main: `git push origin main`

## Commit Types

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `refactor:` code refactoring
- `test:` tests

## Code Style

- Use ESLint
- Follow existing code patterns
- Add comments for complex logic

## Testing

```bash
# Run all tests
npm test

# Run coverage
npm test -- --coverage
```
